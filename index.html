Software Developer & Researcher

ADETTI - MICROSOFT PT - CERN

 

   I have been working at CERN® (European Organization for Nuclear Research) in Switzerland for the last year and a half. On the first half year, I worked as a software developer for the beams section, where I became responsible for the development and integration of a monitoring system in Linux and Android. This system’s main role was to query large amounts of data from the detectors in real-time and then filter all the data, via a fully customizable XML system. This data would then be downloaded on-demand by the users over the Android applications that I developed. Afterwards, I got to work as a Telecommunications & Radio Engineer providing 4th level support for CERN users (3000+ workers) while migrating the old mobile network operator, and correlated equipment, from Sunrise to Swisscom. Afterwards, in order to achieve network stability, coverage and supply high throughputs over the mobile networks technologies: GSM, UMTS and LTE, I became the responsible for the mobile monitoring system's deployment, implementation, sysadmin and dba.

In this project I also had the opportunity to work create real-time statistics analysis tools using JavaScript third party libraries such as: D3, AnyChart, JQuery, etc. In order to supply a user friendly interface for the users, I created a dynamic and long term self-sustainable query system. This system queries in real-time values on-demand (PHP to MSSQL) for each probe on each mobile network technology and then uses this data to create dynamic charts and plot data with a valid GPS location on a map, using Google Maps API. Since one of CERN’s primary goals is data security, I also had to take into account the input system by adding prepared statements, isolating the Webadmin access from the Webroot location and test the code thoroughly.

  Before working at CERN I also worked as a research assistant in ISCTE-IUL - ADETTI® research group which is directly affiliated with Microsoft PT, receiving a Student Grant as a junior Research Assistant. On this project I had the opportunity to work with an interdisciplinary team of both engineers and architects where the common goal was machine learning involving: shape grammars, LISP language implementation and JAVA 7 migration, complex algorithmic calculations, etc.
